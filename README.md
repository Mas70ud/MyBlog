# Personal Blog – دوزبانه (فارسی/انگلیسی) با قابلیت دارک مود

وبلاگ شخصی ساده، واکنش‌گرا و کاملًا سمت کاربر (Client-side) با پشتیبانی از دو زبان فارسی (RTL) و انگلیسی (LTR)، دارای تم روشن/تاریک و پست‌های پویا از فایل‌های JSON.

## ✨ ویژگی‌ها

- دو زبان کامل (فارسی / انگلیسی) – تغییر زبان با حفظ جهت نوشتار
- دارک مود / لایت مود – ذخیره خودکار در مرورگر
- پست‌های پویا – هر پست در یک فایل JSON مجزا (قابل افزودن/حذف)
- بدون دیتابیس – فقط فایل‌های استاتیک
- واکنش‌گرا – مناسب برای موبایل، تبلت و دسکتاپ
- ساختار ماژولار – جداسازی CSS، JS، کامپوننت‌ها برای نگهداری آسان

## 🚀 راه‌اندازی

فایل `index.html` را در مرورگر خود باز کنید.  
(توجه: برای بارگذاری صحیح فایل‌های JSON، بهتر است پروژه روی یک سرور محلی یا هاست اجرا شود.)

## 🛠 شخصی‌سازی

- متن‌های ثابت (نام، بیو، موقعیت، سرویس‌ها و...) را در `js/translations.js` ویرایش کنید.
- لینک شبکه‌های اجتماعی و سرویس‌ها را در `components/bottom-bar.html` تغییر دهید.
- رنگ اصلی (نارنجی `#D95A41`) و سایر متغیرها در `css/base.css` قابل تغییر است.
